# Task UI

Simple task monitoring UI with realtime updates.

## Tech Stack

* React
* WebSocket
* Polling fallback
* Express mock server

## Features

* Task list
* Task detail view
* Status / stage / progress display
* Timeline updates via WebSocket
* API polling fallback

## Run locally

Install dependencies

npm install

Start mock backend

npm run mock-server

Start frontend

npm start

## Production build

npm run build

Serve build

npm install -g serve
serve -s build
